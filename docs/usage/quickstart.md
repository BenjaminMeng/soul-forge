# Soul Forge 快速上手

安装完成后，按以下步骤完成首次校准。

## 第一步：启动问卷

在 Telegram 给你的 bot 发送：

```
/soul-forge
```

Soul Forge 会先展示一段隐私说明，确认你了解数据如何使用。

## 第二步：回答 8 个问题

问卷包含 8 道场景题，每题有 4 个选项（A/B/C/D）。直接回复字母即可。

示例：

```
场景：团队讨论方案时，你更倾向于...
A. 直接说出最优解，不绕弯子
B. 先听听大家怎么说，活跃一下氛围
C. 确认每个人都能接受再推进
D. 梳理清楚每个方案的利弊再决定

你的回复：A
```

整个问卷大约 3 分钟完成。

## 第三步：确认 DISC 类型

问卷结束后，Soul Forge 会展示识别到的 DISC 类型和简要说明。你可以：

- 直接确认（发送 `确认` 或 `yes`）
- 或者告诉它哪里不对，手动调整

## 之后

校准完成后，Soul Forge 会在每次对话结束时静默观察，逐步积累对你沟通风格的理解。不需要再做任何额外操作。

如果你想查看当前状态：

```
/soul-forge status
```

如果想重新校准：

```
/soul-forge reset
```

---

## DISC 类型简介

| 类型 | 特点 | Soul Forge 的适配方向 |
|------|------|----------------------|
| **D** (Dominant) | 直接、目标导向、不喜欢废话 | 简洁、结论前置、减少铺垫 |
| **I** (Influential) | 活跃、喜欢互动、感染力强 | 有温度、适当互动、避免过于冷峻 |
| **S** (Steady) | 耐心、协作、需要稳定感 | 步骤清晰、有确认、避免突然转变 |
| **C** (Conscientious) | 分析型、追求精确、注重细节 | 有逻辑、有依据、减少模糊表述 |

大多数人是混合型，Soul Forge 会识别你的主轴和副轴。
